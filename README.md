# mika-online-compiler

# Introduction:

an editor compile &amp; preview on browser online
这是一个基于前端渲染的在线编辑器，目前支持轻量级少量 react以及vue,文件内容编辑，支持react antDesign,和vue elementPlus组件引入，后续功能逐步完善中
